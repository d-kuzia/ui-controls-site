UI Controls

Повний перелік елементів інтерфейсу з поясненнями та прикладами

1. Radio Button (Радіокнопка)

Опис: Radio button дозволяє користувачу вибрати лише один варіант з групи взаємовиключних опцій. Використовується коли потрібно зробити єдиний вибір з декількох альтернатив.

Приклад:

Вибір

2. Checkbox (Прапорець)

Опис: Checkbox дозволяє користувачу вибрати один або декілька варіантів із списку опцій. Кожен checkbox працює незалежно від інших, що дозволяє множинний вибір.

Приклад:

Вибір багато

3. Text Input (Текстове поле)

Опис: Text input дозволяє користувачу вводити текстові дані. Може мати різні типи для різних видів інформації (текст, email, пароль, число тощо).

Приклад:

Основні поля форми

4. Tabs (Вкладки)

Опис: Tabs дозволяють організувати контент у вкладки, показуючи лише одну секцію за раз. Це економить місце на сторінці та покращує навігацію по різних розділах.

5. Button (Кнопка)

Опис: Button - інтерактивний елемент, який дозволяє користувачу виконати певну дію одним кліком. Може мати різні стилі залежно від важливості дії.

Приклад:

Інтерактивна кнопка

6. Text Label (Текстова мітка)

Опис: Text label використовується для підпису інших елементів форми, надання інструкцій або відображення статичної інформації. Покращує доступність та зрозумілість інтерфейсу.

Приклад:

Мітки для полів форми

7. Link (Посилання)

Опис: Link дозволяє користувачу переходити на інші сторінки, розділи сайту або зовнішні ресурси. Зазвичай виділяється кольором та може мати підкреслення.

Приклад:

Google GitHub

8. Tooltip (Спливаюча підказка)

Опис: Tooltip - це невелике спливаюче вікно з додатковою інформацією, яке з'являється при наведенні курсора на елемент. Використовується для надання контекстної допомоги.

Приклад:

Слово з підказкою

9. Dropdown List (Випадаючий список)

Опис: Dropdown list дозволяє користувачу вибрати один варіант з попередньо визначеного списку опцій. Економить місце на сторінці, показуючи варіанти лише при необхідності.

Приклад:

Вибір

10. Data Grid (Таблиця даних)

Опис: Data grid використовується для відображення структурованих даних у табличному форматі. Дозволяє легко порівнювати та аналізувати інформацію по рядках і стовпцях.
